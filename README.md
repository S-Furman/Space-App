# Space App

Web application that allows to check actual weather on Mars and when International Space Station passes over your location.   

## Tech Stack for this project

- HTML
- CSS
- JavaScript
- React
- React Hooks
- React Router
- API

### Installation

```

$ git clone https://github.com/S-Furman/Space-App.git

$ cd Space-App

$npm install

$npm start
```
